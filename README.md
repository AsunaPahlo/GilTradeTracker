# Gil Trade Tracker

Author: elliegator

A simple Dalamud plugin for FFXIV that tracks gil received from players, keeps a running total, allows entries to be deleted, and exports entries to a CSV file that opens in Excel.

Command:

```text
/giltracker
```

## Features

- Add player name and gil amount manually
- See total gil tracked
- Delete single entries
- Delete all entries
- Export entries to CSV for Excel

## Notes

This first version uses manual entry because automatic gil detection depends on the exact FFXIV chat/event data available in Dalamud and can be added later once the basic plugin builds and loads successfully.
